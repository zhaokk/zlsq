⭐锁定模式
1. 初始状态为堡垒模式，长按设置键3秒可以依次切换到普通定时模式，密码锁模式，无限模式（只有未锁定且没有进入设置时间的状态才能切换模式）
2. 堡垒模式使用独立的一次性密码；普通定时模式和密码锁模式使用同一组主密码和解锁密码
3. 只有普通定时模式和密码锁模式的解锁密码支持修改，主密码不能修改
4. 主密码用于 ①修改解锁密码的前置确认；②在普通定时模式和密码锁模式下，如果忘记解锁密码，可以用主密码来解锁
5. 10秒无操作会自动息屏
6. 开盖检测：当用户按“锁定键”时，如果检测到盖子没有完全合上 → 不进入倒计时，不上锁。1分钟内关闭盖子后自动继续执行 5 秒倒计时（无需重来一次），超过1分钟不合上盖子，取消锁定流程
  1. 按“锁定键”后，如果盖子没有合上，有滴滴声提示用户锁定未完成
  2. 超过1分钟不合上盖子，取消锁定流程时，有滴滴声提示用户锁定已取消
7. 每次切换模式后屏幕上显示的时间都归0

1. 堡垒模式（定时锁定，6 组一次性紧急解锁密码）
锁定
1. 合上盖子
2. 按 任意键 点亮屏幕，检查是否显示盾牌“🛡”（🛡代表堡垒模式）；如果无盾牌 → 长按 设置键 3 秒切换到堡垒模式
3. 按 设置键 循环选位（分钟 → 小时 → 天 ），上下键调时间，最长锁定 400 天
  （如果不想定时，单击 返回键 退出时间设置界面；如果10秒无操作 → 自动退出时间设置界面）
4. 单击 锁定键 确认 ，进入5 秒倒计时（期间可单击返回键可以取消锁定）
  1. 5秒倒计时过程中可以打开盖子重新取放物品，合上盖子后重新开始5秒倒计时
5. 倒计时结束后上锁
6. 锁定时间结束后自动解锁，或用紧急解锁密码提前解锁
紧急解锁（6 组一次性紧急解锁密码）
1. 锁定状态下，双击 返回键 解锁，进入密码输入模式，密码为堡垒模式密码
2. 单击 设置键 循环选位，每数位一跳，上下键调数字（7位数密码，格式 000.00.00 - 999.99.99），单击返回键可退回到锁定倒计时状态
3. 单击 锁定键 确认密码，密码正确→ 立即解锁；错误→ 返回锁定倒计时界面
4. 进入密码输入模式后，10秒无操作，自动退出密码输入模式


2. 普通定时模式（定时锁定，1组紧急解锁密码，可以修改）
锁定
1. 合上盖子
2. 按任意键点亮屏幕，若显示盾牌“🛡”或钥匙“🔑” 或“∞” ，长按 设置键 3 秒依次切换回普通定时模式（该模式下不显示任何图标）
3. 单击 设置键 循环选位，每数位一跳，上下键调时间，最长锁定 400 天
  （如果不想定时，单击 返回键 退出时间设置界面；如果10秒无操作 → 自动退出时间设置界面）
4. 单击 锁定键 确认 ，进入5 秒倒计时（期间单击返回键可以取消锁定），结束后上锁
  1. 5秒倒计时过程中可以打开盖子重新取放物品，合上盖子后重新开始5秒倒计时
5. 倒计时结束后上锁
6. 锁定时间结束后自动解锁，或用紧急解锁密码提前解锁
紧急解锁（1组紧急解锁密码，可以修改）
1. 锁定状态下，双击 返回键 解锁，进入密码输入模式, 密码为普通定时模式密码
2. 单击 设置键 循环选位，每数位一跳，上下键调数字（7位数密码，格式 000.00.00 - 999.99.99），单击返回键可退回到锁定倒计时状态
3. 单击 锁定键 确认密码，密码正确→ 立即解锁；错误→ 返回锁定倒计时界面
4. 进入密码输入模式后，10秒无操作，自动退出密码输入模式


3. 密码锁模式（无定时，屏幕上不显示时间，密码解锁）
锁定
1. 合上盖子
2. 按任意键点亮屏幕，检查是否显示钥匙“🔑”（🔑代表密码锁模式），如果无钥匙 → 长按 设置键 3 秒切换到密码锁模式
3. 不设定时间，双击 锁定键 上锁，进入5秒倒计时（期间单击返回键可以取消锁定）
  1. 5秒倒计时过程中可以打开盖子重新取放物品，合上盖子后重新开始5秒倒计时
4. 倒计时结束后上锁，屏幕上不显示锁定计时
解锁（密码解锁，密码和普通定时模式的密码一致）
1. 锁定状态下，双击 返回键 解锁 → 进入密码输入模式，密码和普通定时模式的一致
2. 单击 设置键 循环选位，每数位一跳，上下键调数字（7位数密码，格式 000.00.00 - 999.99.99），单击返回键可退回到锁定倒计时状态
3. 单击 锁定键 确认密码，密码正确→ 立即解锁；错误→ 返回锁定界面
4. 进入密码输入模式后，10秒无操作，自动退出密码输入模式


4. 无限模式（无定时，从0开始计时，延时解锁）
锁定
1. 合上盖子
2. 按任意键点亮屏幕，检查是否显示“∞”（∞代表密码锁模式），如果无∞ → 长按 设置键 3 秒切换到无限模式
3. 不设定时间，双击 锁定键 上锁，进入5秒倒计时（期间单击返回键可以取消锁定）
  1. 5秒倒计时过程中可以打开盖子重新取放物品，合上盖子后重新开始5秒倒计时
4. 倒计时结束后上锁，屏幕上会从0开始计时
解锁
1. 锁定状态下，双击 返回键 解锁→ 进入 3 分钟“解锁延时”状态，屏幕上“∞”闪烁（期间单击 返回键 可以取消解锁，延时的这段时间也计入锁定时长）
2. 延时结束进入 5 分钟“待解锁”状态，屏幕上锁定图标“🔒”闪烁，此状态下锁舌收回，已解锁
3. 在“待解锁”状态内，打开盖子则代表解锁成功；
4. 如果“待解锁”状态超时不进行任何操作（不开盖），则重新转入无限模式并累计时间（包括延时+待解锁的8分钟）


⭐普通定时模式&密码锁模式的密码设置
1. 在任意模式的未锁定状态下，长按 返回键 10秒 → 进入设置状态，屏幕上显示齿轮“⚙”
2. 输入主密码（7位数密码，格式 000.00.00 - 999.99.99），单击锁定键确认（若单击返回键则退出设置状态）
3. 若主密码输入错误 → 自动退出设置状态
     若主密码输入正确 → 进入更新密码模式：单击 设置键 循环选位，每数位一跳，上下键改数字
4. 密码设置好后，单击 锁定键 保存密码，随后自动退出设置状态，返回初始页面
5. 进入密码输入&更新模式后，10 秒无操作 → 退出设置状态，返回初始页面


⭐记忆功能（自动保存最近5次不同的锁定时长，相同锁定时长只记录一次）
❗只有堡垒模式和普通定时模式下，才能使用记忆功能
1. 合上盖子
2. 按任意键点亮屏幕，长按 设置键 3秒，切换到堡垒模式或普通定时模式
3. 未锁定状态下，按上下键选取前5次锁定时长
4. 若单击 返回键 → 退出记忆功能；若单击 锁定键 → 确认锁定，进入5 秒倒计时（期间单击返回键可以取消锁定）
5. 倒计时结束后上锁
6. 堡垒模式和普通定时模式解锁后，屏幕上会直接显示上一次的锁定时间


⭐锁定中延长锁定时间
❗只有堡垒模式和普通定时模式下，才能使用延长锁定时间功能
1. 在堡垒模式或普通定时模式的锁定状态下，长按 设置键 3秒，进入延长设置，此时屏幕上显示的是剩余锁定时间
2. 单击 设置键 循环选位，每数位一跳，按上下键修改时长（设置的时间只能大于当前锁定的剩余时间）
3. 若单击返回键 → 退出延长；若单击 锁定键 → 延长成功
4. 进入延长设定后，如果10秒无操作 → 退出延长，返回到锁定倒计时页面


⭐儿童锁功能
1. 同时长按 返回键和锁定键 5秒，开启或关闭儿童锁模式。启用儿童锁后，屏幕上会显示儿童图标
2. 在此状态下，除了儿童锁功能按钮外，所有按键操作均无效。按下任意无效键时，屏幕上的儿童图标会闪烁，提示用户已打开儿童锁


⭐重置功能
1. 同时长按 设置键+锁定键 20秒，触发设备重置。重置后设备将恢复出厂设置，包括记忆功能，所有紧急解锁密码，21天打卡数据


⭐21天自律计划（堡垒模式下）
1. 每日自律判定条件：屏幕上有21格打卡进度条，用户每天完成一次堡垒模式下“锁定 → 到时自动解锁”的流程且锁定时间≥30分钟，即视为打卡成功，点亮1个进度格（如果使用紧急密码提前解锁，则打卡失败，不点亮格子），每次打卡间隔要大于18小时
2. 完成奖励和循环机制：当21格全部点亮后，屏幕全屏闪烁，蜂鸣器播放一段短旋律（Gonna Fly Now前8秒）。下一天打卡重新从第一格开始，开启新一轮21天打卡
暂时无法在飞书文档外展示此内容
3. 中断规则：如果连续72小时没有完成一次有效打卡，则当轮 21 天自律计划失败并清零所有进度，下一天打卡从第 1 格重新开始

⭐低电量保护
1. 低电量时，按下锁定键，屏幕上电池图标闪烁，并禁止上锁
